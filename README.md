# Tengxiaofei-git.github.io

腾小飞的 GitHub Pages 静态个人名片示例，内容与结构见仓库根目录的 `goals.md`。

## 技术说明

- 单页：`index.html`，样式通过 [Tailwind CSS](https://tailwindcss.com/) CDN 与内联 `tailwind.config` 定制主题色与暗色模式
- 交互：`script.js`（年份、主题偏好存 `localStorage`、滚动区段与导航高亮联动）
- 字体：Google Fonts「Noto Sans SC」

本地预览可直接用浏览器打开 `index.html`，或通过任意静态服务器托管根目录。
